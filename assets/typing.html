I made a typing speed game using HTML CSS and JavaScript. Typing speed test is a game where you can check your typing speed like WPM (Word Per Minute), CPM (Character Per Minute), Accuracy, etc. In my Typing Speed Test Game, the max time for typing is 60 seconds. Once you start typing, you’ll see your time, mistakes, WPM, and CPM at the bottom. Let's introduce myself and the game at the same time~!?

TYPING SPEED GAME

HTML CSS 및 Vanilla JavaScript를 사용하여 타이핑 스피드 게임을 만들었습니다. 타자 속도 테스트는 WPM(Word Per Minute), CPM(Character Per Minute), 정확도 등 자신의 타자 속도를 확인할 수 있는 게임입니다. 제 타자 속도 테스트 게임의 최대 타자 시간은 60초입니다. 입력을 시작하면 하단에 시간, 실수, WPM 및 CPM이 표시됩니다. 잘못된 문자를 지우거나 키보드의 백스페이스 키를 눌러 뒤로 이동할 수도 있습니다. 모든 문자를 입력했거나 시간이 완료되면 다시 시도 버튼을 클릭하여 결과를 재설정하고 새 단락을 로드할 수 있습니다. 저의 대한 소개와 게임을동시에 하러 가볼까요~!?
